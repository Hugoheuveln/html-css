Mijn HTML probleem!

Voor mijn vierde HTML/CSS les moest ik een probleem vertellen dat ik ervaar met html. Hiervoor heb ik vervolgens een oplossing verzonnen en dit heb ik vervolgens netjes in een stappenplan gezet.

Het probleem

Ik ervaar regelmatig problemen met HTML. Soms kloppen bepaalde CSS afmetingen niet. Soms link in de afbeelding niet goed maar voor mij is het meest voorkomende probleem dat ik eigenlijk geen idee heb wat ik aan het doen ben! Gelukkig is er voor alles een oplossing en ook voor deze ontwetendheid van mij is overal op het internet wel een oplossing voor te vinden. Door het probleem wat ik ervaar te googlen is er eigenlijk altijd wel wat te vinden. Zo kon ik bijvoorbeeld tijdens het maken van deze les niet meer vinden hoe ik een lijst kon maken met cijfers. Ik wist dat "ul" stond voor unlisted list maar ik wist niet wat mij nou cijfers voor mijn zinnen zou geven. Mijn gevolgde stappenplan heb ik hieronder neergezet!

De oplossing!

  1. Ervaar het probleem! in dit geval dus "hoe maak ik een lijst met cijfers"

  2. Bedenk voor jezelf hoe je dit het beste kan opzoeken.

  3. Ga naar google en gebruik duidelijke zoektermen zoals "HTML" en "numbered list"

  4. Vind een goede website. Probeer hierbij bekende HTML websites te gebruiken zoals w3schools.

  5. Kijk op de website hoe je deze oplossing kan toepassen.

  6. Pas de oplossing toe aan je probleem en controleer of het werkt.

  7. Indien het probleem nog aanwezig is. Google verder en kijk in de code waar het fout lijkt te gaan.

Nu is je probleem opgelost en kan je weer blij verder programmeren
